# 📢 Регистрационная воронка

Лендинг для сбора регистраций с отправкой данных в Telegram.

## 🚀 Быстрый старт

### 1. Установка зависимостей
```bash
cd /Users/bambook/воронка
npm install
```

### 2. Настройка Telegram бота

1. Откройте Telegram и найдите @BotFather
2. Отправьте `/newbot` для создания бота
3. Следуйте инструкциям и получите токен
4. Найдите @userinfobot и отправьте `/start` — получите ваш Chat ID

### 3. Настройка .env

Отредактируйте файл `.env`:
```
TELEGRAM_BOT_TOKEN=ваш_токен_бота
ADMIN_CHAT_ID=ваш_chat_id
```

### 4. Запуск

```bash
npm start
```
